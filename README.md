## Candice

![Hello Candice Code Snippet](./screenshots/carbon.png)

Dead simple programming language.
Not expected to be prepared for a real developer environment but can
be fun to work with.

## Philosophy

- Simple syntax.
- 'Low level' capabilities.
- C interoperability.
- Compiled with LLVM and typed.
